GitHub Pages "File Not Found" Solution

If you're seeing a "File not found" error on GitHub Pages, follow these steps to fix it:

1Check File Name

Ensure your main HTML file is named index.html (all lowercase).

2Repository Settings

Go to your GitHub repository Settings → Pages → select main branch as source.

3Wait for Deployment

It can take up to 10 minutes for your site to deploy after changes.

4Check URL

Visit https://yourusername.github.io/repositoryname (all lowercase).
